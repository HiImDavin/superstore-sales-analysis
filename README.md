# Superstore Sales & Profit Analysis

## Latar Belakang
Analisis end-to-end dataset penjualan retail untuk mengidentifikasi 
penyebab profit rendah dan memberikan rekomendasi bisnis berbasis data.

## Tools
- Python (pandas) — data cleaning & EDA
- Power BI — dashboard visualisasi

## Key Insights
- Technology adalah kategori paling profitable (margin 17.4%)
- Furniture hanya margin 2.5% — Sub-kategori Tables rugi $17,725
- Diskon di atas 30% menyebabkan kerugian rata-rata $156/transaksi
- Total kerugian akibat diskon berlebihan: $135,364
- Region West paling profitable, Central paling banyak kasih diskon besar

## Rekomendasi Bisnis
1. Batasi diskon maksimal 30% untuk semua kategori
2. Audit struktur biaya Tables — investigasi harga modal dan biaya pengiriman
3. Replikasi strategi Region West ke region lain
